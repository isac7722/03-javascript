02. 태그 이름을 이용한 요소 노드 획득

Document.prototype/Element.prototype.getElementByTagName ("태그") 메서드는 인수로 전달한 태그 이름을 갖는 모든 요소 노드를 탐색하요 반환한다. 메서드 이름의 Elements가 복수형인 것에서 알 수 있듯이 여러 개의 요소 노드 객체를 갖는 dom 컬렉션 객체인 htmlCollection 객체를 반환한다

  • burger
  • rice
  • ham
  • water
  • cola
  • sprite
  • lion
  • tiger
  • lamb
  • giraffe
